# SVG-Logo-Maker


## Description

Greetingns! Welcome to my SVG logo maker project! This project will allow users to create a SVG logo maker with options to choose from, from the text inside the shape, the color via name or hex, choosing the shape and choosing the color of the shape via hex code or name!



## Installation

1. Clone the repo
2. Open in VS code
3. Install node.js
4. Once node.js is installed, type npm init in the terminal to create your package.json files
5. Type in npm install to instal your dependencies
6. Make sure youre in the right index.js file, and run node index.js
7. Have fun!
